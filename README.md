# Тестирование устойчивости чат-бота зоопарка к adversarial-атакам

> Тестовое задание по adversarial ML, prompt injection и RAG robustness

Цель — проверить устойчивость RAG-чат-бота зоопарка к вредоносным пользовательским запросам. Бот отвечает на вопросы о ценах, льготах, расписании и животных. Неверные ответы о льготах и ценах несут прямой финансовый и юридический риск для организации.

---

## Задания

### Задание 1 — Adversarial-промпты и классификатор атак
Разработан классификатор из 10 классов атак: Prompt Injection, Jailbreak, ценовые манипуляции, галлюцинации, текстовые искажения, многоязычность, impersonation, RAG injection, опасный контент, context DoS.

Составлено 10 уникальных базовых промптов с обоснованием цели атаки и ожидаемого поведения бота. Дополнительно разработано 5 многошаговых сценариев (3–5 реплик) с постепенной эскалацией давления.

### Задание 2 — Синтез 200 вариаций запросов
На основе 10 базовых промптов синтезировано 200 вариаций методом LLM-assisted paraphrase augmentation по 4 стратегиям:
- **P** — Paraphrase (перефразирование, 70 шт.)
- **S** — Style (смена стиля подачи, 50 шт.)
- **D** — Distortion (текстовые искажения, 40 шт.)
- **C** — Context (смена роли пользователя, 40 шт.)

Каждая вариация размечена по уровню сложности: Simple / Medium / Hard.

### Задание 3 — Обучающая выборка для LoRA-адаптера
Сформирована выборка 1050+ примеров в формате Alpaca instruction-following. Источники: LLM-синтез (~700), публичные датасеты HuggingFace (~150), ручная разметка (~100), аугментация из задания 2 (200). Балансировка по 10 классам атак и 3 уровням сложности.

### Задание 4* — LoRA-адаптер (класс C5: текстовые искажения)
Выбрана базовая модель **dolphin-2.9-llama3-8b** (нецензурированный fine-tune LLaMA3 8B). Схема обучения: QLoRA 4-bit, r=16, lora_alpha=32, 3 эпохи, effective batch size 32.

Реализован детерминированный генератор adversarial-запросов (seed=42) с 5 подтипами искажений. Сгенерировано 100 примеров с 100% уникальностью и равномерным распределением по подтипам.
