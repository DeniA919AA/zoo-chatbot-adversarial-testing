# Тестирование устойчивости чат-бота зоопарка к adversarial-атакам

Тестовое задание по adversarial ML и prompt injection для RAG-чат-бота зоопарка.

---

## Структура репозитория

```
├── zoo_chatbot_test_report.pdf    # Основной отчёт (задания 1–4)
├── zoo_chatbot_dataset.xlsx       # Датасеты заданий 1, 2, 3
└── task4/
    ├── train.py                   # Код обучения LoRA-адаптера
    ├── generate.py                # Код генерации adversarial-датасета
    ├── config.json                # Гиперпараметры обучения
    ├── metrics.json               # Метрики качества датасета
    ├── examples.json              # Примеры трансформаций до/после
    ├── adversarial_dataset_c5_real.xlsx  # Реальный датасет (100 примеров)
    ├── adversarial_results.json   # Датасет в формате JSON
    └── README.md                  # Инструкция по запуску задания 4
```

---

## Задания

| Задание | Описание |
|---------|----------|
| 1 | 10 adversarial-промптов, классификатор атак, 5 многошаговых сценариев |
| 2 | 200 вариаций запросов (Paraphrase / Style / Distortion / Context) |
| 3 | Обучающая выборка 1050+ примеров для LoRA-адаптера |
| 4* | Детерминированный генератор adversarial-запросов класса C5 |
